YouTube Vibe Check reads the URL of the currently active YouTube tab when the user opens the extension. The extension extracts the YouTube video ID from that URL & sends it to our backend service to retrieve public YouTube metadata such as title, channel name, publish date, duration, views, likes & comments. This information is used only to generate the extension’s rating card and related signals, including watch signal, crowd pulse, time fit & freshness.

The extension uses temporary browser session storage to cache video metadata for the current tab while the session is active. The extension does not require account creation & does not sell personal data.

Requests may be processed by our hosting provider and by Google’s YouTube Data API in order to return the requested video metadata. Standard server logs may be generated by those services as part of normal request handling.

We use the data only to provide the extension’s core functionality. We do not use the data for advertising, creditworthiness, or unrelated profiling.
